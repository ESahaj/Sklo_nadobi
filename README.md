# Hra: Sklo a nádobí – Chemie

Tato jednoduchá webová hra slouží k procvičování a poznávání chemického skla a laboratorního nádobí.  
Je určena pro studenty chemie a je součástí školního projektu.

## Jak hrát

- Hra zobrazí obrázek skla/nádobí  
- Vaším úkolem je vybrat správný název  
- Body se počítají automaticky  
- Hra končí, když projdete všechny položky

## Spuštění hry online

Hru si můžete zahrát zde: [Spustit hru](https://ESahaj.github.io/sklo_nadobi/)

- HTML, CSS, JavaScript
